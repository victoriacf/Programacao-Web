Lista Ordenada

A lista ordenada é representada pela tag <ol> e cada item de lista será representada pela tag <li>
Item List. Resultado:

  1. Item A
  2. Item B
  3. Item C

Lista Não Ordenada

A lista não ordenada é representada pela tag <ul> e cada item de lista será representada pela tag <li>
Item List. Resultado:

  • Item A
  • Item B
  • Item C
